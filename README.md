# HyattButwal.com

HyattButwal.com is a premium luxury domain landing page built with a cinematic modern design style.

The project combines clean hospitality branding with a Netflix-inspired user interface to create a strong visual presentation for a premium digital asset sale.

Built using pure HTML, CSS, and JavaScript, the website focuses on smooth animations, responsive layouts, dark luxury aesthetics, and modern UI interactions.

## Features

* Responsive modern layout
* Cinematic hero section
* Animated scrolling poster rows
* Luxury dark theme
* Mobile-friendly design
* Smooth hover animations
* SEO and social media meta tags
* Fast lightweight structure
* Pure HTML, CSS, and JavaScript

## Design Style

The interface is inspired by:

* Luxury hotel branding
* Modern streaming platforms
* Premium entertainment websites
* Nepal tourism visuals

## Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
* Google Fonts
* Unsplash Images

## Purpose

This project was created as:

* A premium domain sale landing page
* A luxury hospitality brand concept
* A cinematic web design showcase
* A modern UI experiment
